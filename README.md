# PasswordHashing

## Here, passwords are hashed with BCrypt and stored in the database.
## validating user input, such as username length and email format.
## Provides an API endpoint to register users, storing hashed passwords securely.
## Verifies login credentials by comparing the input password with the stored hashed password using BCrypt.
## Custom exception classes and IExceptionHandler is implemented in AppExceptionHandler Class for meaningful error message display.
## Implemented DTO, Mapper for Data transfer from Model.
## Repository pattern, service layer are used for database interaction and business logic respectively.
